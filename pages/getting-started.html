快速入门

3 步开始使用 BMAD

首页 / 快速入门

在几分钟内安装并运行 BMAD，开始体验 AI 驱动的敏捷开发。

第一步：安装 BMAD

打开终端，在项目目录中运行：

npx bmad-method install

安装器会引导你选择模块。选择 BMad Method (BMM) 作为核心模块。

安装完成后，会创建两个文件夹：

  • _bmad/ — agents、workflows、tasks 和配置文件
  • _bmad-output/ — 产出文件目录

第二步：打开 AI IDE

在项目文件夹中启动你喜欢的 AI IDE：

Claude Code Cursor Windsurf Cline

第三步：开始使用

在 AI IDE 中运行：

/bmad-help

BMad-Help 会检测你的项目状态，告诉你下一步做什么。

试试问：

"我有一个 SaaS 产品的想法，应该从哪里开始？"

理解四阶段流程

BMAD 通过四个阶段帮助你构建软件：

1

Analysis (可选)

头脑风暴、研究、产品简报

2

Planning (必须)

创建 PRD 或 Tech-spec

3

Solutioning (Method/Enterprise)

设计架构，分解 Epics 和 Stories

4

Implementation

Sprint 开发，循环迭代

常用命令

命令 Agent 用途
/bmad-help Any 智能向导，问任何问题
/bmad-bmm-create-prd PM 创建产品需求文档
/bmad-bmm-create-architecture Architect 创建架构设计文档
/bmad-bmm-create-epics-and-stories PM 分解 Epics 和 Stories
/bmad-bmm-sprint-planning SM 初始化 Sprint 追踪
/bmad-bmm-dev-story DEV 开发 Story
